Canadian OpenStack Users


Individuals

Probably the best list, right now, of individuals using OpenStack in Canada can be found on the #canstack meetup group.

Organizations

These are a few of the organizations and companies that are using OpenStack in Canada. If your organization is not on this list and you would like it to be, or vice-versa, please let us know. It's not a complete list.

Canarie DAIR program

The Canarie DAIR DAIR program runs OpenStack.

CloudA

Cloud A is a leading provider of Cloud Infrastructure powered by Openstack based in Canada. Our product simplifies the installation and management of the hardware and software that provides the infrastructure for large scale environments having hundreds or thousands of servers supporting Big Data, Analytics, or High Performance Computing. For more information visit www.CloudA.ca.

Cybera

Cybera is a not-for-profit agency that accelerates technology adoption in Alberta.

eNovanace

eNovance is the European leader and a major global technology player in open cloud computing software with offices in Montreal.

Gridcentric

Gridcentric's products integrate with OpenStack.

University of Alberta Library

The UofA library uses OpenStack Swift as part of their collection storage system.

Canadian Web Hosting

Canadian Web Hosting uses OpenStack to provide on-demand computing resources.

VMFarms - nibbler.io

VMFarms provides an object storage system based on OpenStack Swift at Nibbler.io.
